# Episode 04 Production Package — Strategy

## Purpose
This package supports the episode transcript about:
- when Strategy is worth it
- identity coupling vs interface coupling
- moving the decision into the right place
- using a factory to isolate decision logic
- improved testability through smaller, more focused tests

## Files
- `conditional_before.py` — original conditional-based design
- `strategy_after.py` — strategy-based refactor with factory
- `test_episode_04.py` — runnable pytest tests
- `UML_before_after.md` — conceptual before/after diagrams
- `shot_list.md` — screen capture plan and effects
- `test_output.txt` — pytest results

## Run
```bash
pytest -q
```

## Capture notes
- Use VS Code dark theme
- Zoom so code is approximately 30–34px equivalent
- No scrolling during a shot
- Use static screenshots or frozen clips
- Hold 2 seconds before/after each shot
- Highlight conditionals first, then creation/decision movement

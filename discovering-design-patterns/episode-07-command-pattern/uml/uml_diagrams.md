# UML Diagrams

SVG images are included in this folder. You can import them into Premiere, Keynote, PowerPoint, or convert to PNG.

## Optional conversion to PNG
```bash
# macOS with Inkscape installed
inkscape command_pattern.svg --export-filename=command_pattern.png
inkscape proxy_pattern.svg --export-filename=proxy_pattern.png
inkscape adapter_pattern.svg --export-filename=adapter_pattern.png
inkscape decorator_pattern.svg --export-filename=decorator_pattern.png
inkscape mock_object.svg --export-filename=mock_object.png
```

## Canonical comparison purpose
Flash these quickly during the line:

> This superficially resembles other design patterns, like Proxy, Adapter, Decorator, or even Mocks, because all of them involve wrapping or delegating to some other object. But the intent is completely different.

Use them as quick visual contrast cards, not teaching diagrams.
